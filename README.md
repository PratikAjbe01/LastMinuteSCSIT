### Setup .env file

```bash
MONGO_URI=
PORT=
JWT_SECRET=
NODE_ENV=development

SMTP_HOST=
SMTP_PORT=
SMTP_MAIL=
SMTP_PASSWORD=

CLIENT_URL= http://localhost:5173
```

### Run this app locally

```shell
npm run build
```

### Start the app

```shell
npm run start
```
